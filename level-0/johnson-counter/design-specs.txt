Design Name: johnson_counter4
Module Name:
- johnson_counter4

Inputs:
- clk        // Clock input
- rst        // Synchronous reset (active high)

Outputs:
- q[3:0]     // Output vector representing internal state

Design Signature:

module johnson_counter4 (
    input clk,
    input rst,
    output [3:0] q
);

Design Notes:
- Initialize `q` to 4'b0000.
- On each clock edge, shift left and feed back inverted MSB to LSB.
- State sequence: 0000 → 0001 → 0011 → 0111 → 1111 → 1110 → 1100 → 1000 → 0000 …
- Loop indefinitely through this sequence.
- Suitable for timing or stepwise control logic with simple decoding.
